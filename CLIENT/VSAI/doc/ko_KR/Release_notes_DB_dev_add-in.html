IBM Database Add-Ins for Visual Studio

DB2 Bondi 릴리스 10.5.0.5부터 Microsoft Visual Studio 2008을 사용한 IBM 데이터 서버 제품 지원은 더 이상 사용되지 않습니다.

릴리스 정보: 버전 11.1 수정 1, 수정팩 1

IBM Database Add-Ins for Visual Studio, 버전 11.1 수정 1, 수정팩 1에서는 IBM Data Server Provider for .NET 및 IBM DB2 Data Provider for .NET Framework 2.0 이상을 지원합니다.

필수 소프트웨어

IBM Database Add-Ins for Visual Studio, 버전 11.1 수정 1, 수정팩 1의 전제조건에는 다음이 포함됩니다.

  • Microsoft Visual Studio 2010, Microsoft Visual Studio 2012, Visual Studio 2013, Visual Studio 2015 또는 Visual Studio 15
  • IBM Data Server Provider for .NET 또는 IBM DB2 Data Provider for .NET Framework 2.0 이상
  • Windows 7, 서비스팩 2, 이상; Windows Server 2008, 서비스팩 2; 또는 Windows Server 2008 R2; Windows 8 이상; Windows Server 2012; Windows 10.

주:  Data Provider 버전마다 다른 버전의 Visual Studio를 지원합니다. Add-in은 해당하는 버전의 Data Provider가 설치되어 있지 않는 경우 설치되어 있는 Visual Studio 버전에 설치되지 않습니다.

  • .NET Framework 2.0용 IBM DB2 Data Provider:
    • Visual Studio 2008
  • .NET 2.0, 3.0 또는 3.5용 IBM Data Server Provider:
    • Visual Studio 2008
  • .NET 4.0 또는 4.5용 IBM Data Server Provider:
    • Visual Studio 2010
    • Visual Studio 2012
    • Visual Studio 2013
    • Visual Studio 2015
    • Visual Studio 15

IBM Database Add-Ins for Visual Studio, 버전 11.1 수정 1, 수정팩 1의 주요 기능

IBM Database Add-Ins for Visual Studio, 버전 11.1 수정 1, 수정팩 1의 새로운 기능을 사용하면 개발자가 Microsoft Visual Studio를 통합 개발 환경으로 사용하여 개발 시간을 줄이고 전체 DB2 서버 제품군 및 Informix® Dynamic Server의 .NET 애플리케이션을 개발할 수 있습니다. 새로운 기능은 다음과 같습니다.

  • Visual Studio 15 지원

IBM Database Add-Ins for Visual Studio, 버전 11.1 수정 1, 수정팩 1은 다음 데이터베이스 서버를 지원합니다.

  • Linux, UNIX 및 Windows용 DB2 버전 9.7 이상
  • z/OS용 DB2 버전 10 이상
  • DB2 for i V6R1, V7R1 및 V7R2
  • Informix Dynamic Server, 버전 11.5, 버전 11.70 및 버전 12.10
사용 가능한 기능과 데이터 유형 호환성은 사용자의 데이터 서버에 따라 다릅니다. 예를 들어, Informix Dynamic Server는 XML 데이터 유형을 지원하지 않습니다.

주:  IBM Database Add-Ins for Visual Studiod와 클라이언트 패키지의 버전, 릴리스 및 수정팩 레벨이 일치하지 않는 경우 문제점이 발생할 수 있습니다. 도구 기능이 올바르게 작동하는지 확인하려면 다음 클라이언트 패키지 중 하나를 동일한 레벨의 Add-in 버전, 릴리스 및 수정팩과 함께 설치하십시오.

  • IBM Data Server Client
  • IBM Data Server Runtime Client
  • IBM Database Add-Ins용 IBM Data Server Driver Package

지원

IBM Database Add-Ins for Visual Studio, 버전 11.1 수정 1, 수정팩 1에 대한 지원은 developerWorks® .NET 포럼에서 제공됩니다. 이 포럼은 IBM Database Add-Ins for Visual Studio, IBM Data Server Provider for .NET 및 IBM DB2 Data Provider for .NET Framework 2.0 이상의 개발자가 모니터합니다. IBM 개발팀은 이후에 릴리즈될 Visual Studio 및 .NET용 IBM Data Provider 및 Add-in 도구에 대한 사용자의 피드백 및 모든 제안사항을 포함할 수 있도록 노력하고 있습니다.

사용자는 developerWorks .NET 포럼에 참여하여 IBM 개발자에게 피드백을 제공할 수 있습니다.
