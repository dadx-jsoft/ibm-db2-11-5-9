IBM Database Add-Ins for Visual Studio, Version 10.1, Fix Pack 2 supports the IBM Data Server Provider for .NET and the IBM DB2 Data Provider for .NET Framework 2.0.

Prerequisites

Prerequisites for the IBM Database Add-Ins for Visual Studio, Version 10.1, Fix Pack 2 include:

  • Microsoft Visual Studio 2008, Microsoft Visual Studio 2010, or Microsoft Visual Studio 2012
  • IBM Data Server Provider for .NET or IBM DB2 Data Provider for .NET Framework 2.0
  • Windows XP, Service Pack 2, or later; Windows Vista, Service Pack 1; Windows 7; Windows Server 2003, Service Pack 2, or later; Windows Server 2008, Service Pack 2; or Windows Server 2008 R2.

Note:  Different versions of the data providers support different versions of Visual Studio. The add-ins are not installed for an installed version of Visual Studio unless a corresponding version of a data provider is installed.

  • IBM DB2 Data Provider for .NET Framework 2.0:
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 2.0, 3.0, or 3.5:
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 4.0, or 4.5:
    • Visual Studio 2010
    • Visual Studio 2012

Key new features in IBM Database Add-Ins for Visual Studio, Version 10.1, Fix Pack 2

The new features in the IBM Database Add-Ins for Visual Studio, Version 10.1, Fix Pack 2 enable developers to reduce development time and develop .NET applications for the entire DB2 server family, as well as for Informix® Dynamic Server, by using Microsoft Visual Studio as their integrated development environment. The following list summarizes the new features:

  • Support for Visual Studio 2012
  • Support for .NET Framework 4.5
  • Support for Entity Framework 4.3

The IBM Database Add-Ins for Visual Studio, Version 10.1, Fix Pack 2 supports the following database servers:

  • DB2 Universal Database™ for Linux, UNIX, and Windows Version 8.2
  • DB2 for Linux, UNIX, and Windows Version 9.1, and later
  • DB2 Universal Database for z/OS Version 8
  • DB2 for z/OS Version 9.1 and Version 10
  • DB2 for i V5R4, V6R1, and V7R1
  • Informix Dynamic Server, Version 11.1, Version 11.5, and Version 11.70
Availability of features and data type compatibility depends on the data server that you use. For example, Informix Dynamic Server does not support the XML data type.

Note:  Problems can occur when the version, release, and fix-pack level of the IBM Database Add-Ins for Visual Studio and a client package are not identical. To ensure that tooling functions work correctly, install the same version, release, and fix-pack level of the add-ins and one of the following client packages:

  • IBM Data Server Client
  • IBM Data Server Runtime Client
  • IBM Data Server Driver Package for IBM Database Add-Ins
