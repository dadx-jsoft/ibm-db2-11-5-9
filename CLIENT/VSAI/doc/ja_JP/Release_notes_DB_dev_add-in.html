IBM Database Add-Ins for Visual Studio

DB2 Bondi リリース 10.5.0.5 以降では、Microsoft Visual Studio 2008 による IBM データ・サーバー製品のサポートが非推奨になっています。

リリース・ノート: バージョン 11.1 モディフィケーション 1 フィックスパック 1

IBM Database Add-Ins for Visual Studio バージョン 11.1 モディフィケーション 1 フィックスパック 1 は、 IBM Data Server Provider for .NET と IBM DB2 Data Provider for .NET Framework 2.0 以降に対応しています。

前提条件

IBM Database Add-Ins for Visual Studio バージョン 11.1 モディフィケーション 1 フィックスパック 1 の前提条件を以下に示します。

  • Microsoft Visual Studio 2010、Microsoft Visual Studio 2012、Visual Studio 2013、Visual Studio 2015、Visual Studio 15 のいずれか
  • IBM Data Server Provider for .NET または IBM DB2 Data Provider for .NET Framework 2.0 以降
  • Windows 7 (Service Pack 2 以降)、Windows Server 2008 (Service Pack 2 以降)、Windows Server 2008 R2、Windows 8 以降、Windows Server 2012、Windows 10 のいずれか

注:  データ・プロバイダーのバージョンによって、サポートされる Visual Studio のバージョンはそれぞれ異なります。 インストールされているバージョンの Visual Studio 用のアドインは、 対応するバージョンのデータ・プロバイダーがインストールされていない限り、インストールされません。

  • IBM DB2 Data Provider for .NET Framework 2.0:
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 2.0、3.0、または 3.5:
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 4.0 または 4.5:
    • Visual Studio 2010
    • Visual Studio 2012
    • Visual Studio 2013
    • Visual Studio 2015
    • Visual Studio 15

IBM Database Add-Ins for Visual Studio バージョン 11.1 モディフィケーション 1 フィックスパック 1 の主要機能

IBM Database Add-Ins for Visual Studio バージョン 11.1 モディフィケーション 1 フィックスパック 1 の新機能により、 開発者は、統合開発環境として Microsoft Visual Studio を使用することで開発時間を短縮し、 すべての DB2 サーバー・ファミリー、および Informix® Dynamic Server 用の .NET アプリケーションを開発できます。 その新機能の要約を以下にリストします。

  • Visual Studio 15 のサポート

IBM Database Add-Ins for Visual Studio バージョン 11.1 モディフィケーション 1 フィックスパック 1 は、以下のデータベース・サーバーに対応しています。

  • DB2 for Linux, UNIX, and Windows バージョン 9.7 以降
  • DB2 for z/OS バージョン 10 以降
  • DB2 for i V6R1、V7R1、および V7R2
  • Informix Dynamic Server バージョン 11.5、バージョン 11.70、およびバージョン 12.10
機能の可用性やデータ・タイプの互換性は、使用するデータ・サーバーによって異なります。例えば、Informix Dynamic Server は XML データ・タイプをサポートしません。

注: IBM Database Add-Ins for Visual Studio とクライアント・パッケージのバージョン、リリース、およびフィックスパックのレベルが同じでない場合、問題が発生する場合があります。 ツールの機能が確実に正しく動作するようにするには、インストールするアドインと以下のいずれかのクライアント・パッケージを、同じバージョン、リリース、およびフィックスパックのレベルにしてください。

  • IBM Data Server Client
  • IBM Data Server Runtime Client
  • IBM Data Server Driver Package for IBM Database Add-Ins

サポート

IBM Database Add-Ins for Visual Studio バージョン 11.1 モディフィケーション 1 フィックスパック 1 のサポートは、developerWorks® .NET forum で提供されています。このフォーラムは、IBM Database Add-Ins for Visual Studio、IBM Data Server Provider for .NET、および IBM DB2 Data Provider for .NET Framework 2.0 以降の開発者によってモニターされています。 この IBM 開発者のチームは、ユーザーと対話することに関心を持っており、 Visual Studio と .NET 用の IBM データ・プロバイダーおよびアドイン・ツールの 次期ドロップおよびリリースに組み込む、あらゆるフィードバックおよび提案に 非常に興味を持っています。

IBM 開発者とのコミュニケーションを図り、有意義なフィードバックを行うために、developerWorks.NET forum に積極的に参加してください。
