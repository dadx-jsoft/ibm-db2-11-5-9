IBM Database Add-Ins for Visual Studio

從 DB2 Bondi 版次 10.5.0.5 起，IBM 資料伺服器產品搭配 Microsoft Visual Studio 2008 的支援已棄用。

版本注意事項：11.1 版修正 1 Fix Pack 1

IBM Database Add-Ins for Visual Studio 11.1 版修正 1 Fix Pack 1 支援 IBM Data Server Provider for .NET 與 IBM DB2 Data Provider for .NET Framework 2.0 及更新版本。

必備項目

IBM Database Add-Ins for Visual Studio 11.1 版修正 1 Fix Pack 1 的必備項目包括：

  • Microsoft Visual Studio 2010、Microsoft Visual Studio 2012、Visual Studio 2013、Visual Studio 2015 或 Visual Studio 15
  • IBM Data Server Provider for .NET 或 IBM DB2 Data Provider for .NET Framework 2.0 及更新版本
  • Windows 7 Service Pack 2 或更新版本、Windows Server 2008 Service Pack 2 或 Windows Server 2008 R2、Windows 8 或更新版本、Windows Server 2012；Windows 10。

附註：不同資料提供者的版本支援不同版本的 Visual Studio。 除非已安裝資料提供者相對應的版本，才能針對 Visual Studio 的版本進行附加程式的安裝。

  • IBM DB2 Data Provider for .NET Framework 2.0：
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 2.0, 3.0, 或 3.5：
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 4.0 或 4.5：
    • Visual Studio 2010
    • Visual Studio 2012
    • Visual Studio 2013
    • Visual Studio 2015
    • Visual Studio 15

IBM Database Add-Ins for Visual Studio 11.1 版修正 1 Fix Pack 1 的主要特性

IBM Database Add-Ins for Visual Studio 11.1 版修正 1 Fix Pack 1 的新增特性可讓開發人員減少開發的時間，並藉由使用 Microsoft Visual Studio 作為其整合的開發環境來為整個 DB2 伺服器系列，以及 Informix® Dynamic Server 開發 .NET 應用程式。 新增特性彙總如下：

  • Visual Studio 15 的支援

IBM Database Add-Ins for Visual Studio 11.1 版修正 1 Fix Pack 1 支援下列資料庫伺服器：

  • DB2 for Linux, UNIX, and Windows 9.7 版及更新版本
  • DB2 for z/OS 第 10 版及更新版本
  • DB2 for i V6R1、V7R1 及 V7R2
  • Informix Dynamic Server 11.5 版、11.70 版及 12.10 版
特性和資料類型相容性的可用性視使用的資料伺服器而定。例如，Informix Dynamic Server 不支援 XML 資料類型。

附註：  如果 IBM Database Add-Ins for Visual Studio 及用戶端套件的版本、版次及修正程式層次不相同，則可能會出現問題。 為確保工具功能可以正確地運作，請安裝相同版本、版次及修正程式層次的附加程式，並安裝下列其中一個用戶端套件：

  • IBM Data Server Client
  • IBM Data Server Runtime Client
  • IBM Data Server Driver Package for IBM Database Add-Ins
