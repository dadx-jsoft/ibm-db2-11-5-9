IBM Database Add-Ins for Visual Studio

Ab DB2 Bondi Release 10.5.0.5 wird Microsoft Visual Studio 2008 für IBM Datenserverprodukt nicht weiter unterstützt.

Releaseinformationen: Version 11.1 Modifikation 1, Fixpack 1

IBM Database Add-Ins for Visual Studio Version 11.1 Modifikation 1 Fixpack 1 unterstützt IBM Data Server Provider for .NET und IBM DB2 Data Provider for .NET Framework 2.0 und höher.

Voraussetzungen

Für IBM Database Add-Ins for Visual Studio Version 11.1 Modifikation 1 Fixpack 1 gelten u. a. die folgenden Voraussetzungen:

  • Microsoft Visual Studio 2010, Microsoft Visual Studio 2012, Visual Studio 2013, Visual Studio 2015 oder Visual Studio 15
  • IBM Data Server Provider for .NET oder IBM DB2 Data Provider for .NET Framework 2.0 und höher
  • Windows 7, Service Pack 2 oder höher; Windows Server 2008, Service Pack 2; oder Windows Server 2008 R2; Windows 8 oder höher; Windows Server 2012; Windows 10.

Anmerkung: Die verschiedenen Versionen der Datenprovider unterstützen unterschiedliche Versionen von Visual Studio. Die Add-ins werden für eine installierte Version von Visual Studio nur installiert, wenn eine entsprechende Version eines Datenproviders installiert ist.

  • IBM DB2 Data Provider for .NET Framework 2.0:
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 2.0, 3.0 oder 3.5:
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 4.0 oder 4.5:
    • Visual Studio 2010
    • Visual Studio 2012
    • Visual Studio 2013
    • Visual Studio 2015
    • Visual Studio 15

Schlüsselfeatures in IBM Database Add-Ins for Visual Studio Version 11.1 Modifikation 1 Fixpack 1

Mithilfe der neuen Features in IBM Database Add-Ins for Visual Studio Version 11.1 Modifikation 1 Fixpack 1 können Entwickler die Entwicklungszeit verkürzen und .NET-Anwendungen für die gesamte Produktfamilie von DB2 Server sowie für Informix® Dynamic Server entwickeln, indem sie Microsoft Visual Studio als integrierte Entwicklungsumgebung verwenden. Die neuen Features sind in der folgenden Liste zusammengefasst:

  • Unterstützung für Visual Studio 15

IBM Database Add-Ins for Visual Studio Version 11.1 Modifikation 1 Fixpack 1 unterstützt die folgenden Datenbankserver:

  • DB2 for Linux, UNIX, and Windows Version 9.7 und höher
  • DB2 for z/OS Version 10 und höher
  • DB2 for i V6R1, V7R1 und V7R2
  • Informix Dynamic Server, Version 11.5, Version 11.70 und Version 12.10
Die Verfügbarkeit von Features und die Kompatibilität von Datentypen hängt von dem von Ihnen verwendeten Datenserver ab. Informix Dynamic Server beispielsweise unterstützt den XML-Datentyp nicht.

Anmerkung:  Wenn die Version, das Release und das Fixpack von IBM Database Add-Ins for Visual Studio und ein Clientpaket nicht denselben Stand haben, können Probleme auftreten. Installieren Sie daher denselben Versions-, Release- und Fixpack-Stand der Add-Ins und eines der folgenden Clientpakete, um sicherzustellen, dass die Toolfunktionen ordnungsgemäß funktionieren:

  • IBM Data Server Client
  • IBM Data Server Runtime Client
  • IBM Data Server Driver Package for IBM Database Add-Ins
