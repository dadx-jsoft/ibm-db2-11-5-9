IBM Database Add-Ins for Visual Studio V11.1.1 FP1 支持 IBM Data Server Provider for .NET 和 IBM DB2 Data Provider for .NET Framework 2.0 以及更高版本。

必备软件

IBM Database Add-Ins for Visual Studio V11.1.1 FP1 包括：

  • Microsoft Visual Studio 2010、Microsoft Visual Studio 2012、Visual Studio 2013、Visual Studio 2015 或 Visual Studio 15
  • IBM Data Server Provider for .NET 或 IBM DB2 Data Provider for .NET Framework 2.0 或更高版本
  • Windows 7 Service Pack 2 或更高版本；Windows Server 2008 Service Pack 2 或 Windows Server 2008 R2；Windows 8 或更高版本；Windows Server 2012；Windows 10。

注：数据提供程序的不同版本支持 Visual Studio 的不同版本。不会为 Visual Studio 的已安装版本安装这些加载件，除非安装了数据提供程序的对应版本。

  • IBM DB2 Data Provider for .NET Framework 2.0：
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 2.0, 3.0, or 3.5：
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 4.0 或 4.5：
    • Visual Studio 2010
    • Visual Studio 2012
    • Visual Studio 2013
    • Visual Studio 2015
    • Visual Studio 15

IBM Database Add-Ins for Visual Studio V11.1.1 FP1 的的主要新功能

通过将 Microsoft Visual Studio 用作其集成开发环境 IBM Database Add-Ins for Visual Studio V11.1.1 FP1 中的新功能使开发者能够减少开发时间，并为整个 DB2 服务器系列以及 Informix® Dynamic Server 开发 .NET 应用程序。 以下列表总结了新功能：

  • 支持 Visual Studio 15

IBM Database Add-Ins for Visual Studio V11.1.1 FP1 支持以下数据库服务器：

  • DB2 for Linux, UNIX, and Windows V9.7 和更高版本
  • DB2 for z/OS V10 和更高版本
  • DB2 for i V6R1、V7R1 和 V7R2
  • Informix Dynamic Server V11.5、V11.70 和 V12.10
功能部件的可用性以及数据类型兼容性取决于您使用的数据服务器。例如，Informix Dynamic Server 不支持 XML 数据类型。

注意：当 IBM Database Add-Ins for Visual Studio 和客户机软件包的版本、发行版以及修订包级别不相同时，会发生问题。要确保工具功能可以正确工作，请安装具有相同版本、发行版以及修订包级别的加载件和以下某个客户机软件包：

  • IBM 数据服务器客户机
  • IBM 数据服务器运行时客户机
  • 用于 IBM Database Add-Ins 的 IBM 数据服务器驱动程序包
