IBM Database Add-Ins for Visual Studio

Para DB2 Bondi Release 10.5.0.5 en adelante, el soporte para productos de servidor de datos de IBM con Microsoft Visual Studio 2008 está en desuso.

Notas del release: Versión 11.1 Mod 1, Fixpack 1

© Copyright International Business Machines Corporation 2002, 2016. Reservados todos los derechos.

IBM Database Add-Ins para Visual Studio, versión 11.1 Mod 1, Fixpack 1 admite IBM Data Server Provider para .NET e IBM DB2 Data Provider for .NET Framework 2.0 y posteriores.

Requisitos previos

Entre los requisitos previos para IBM Database Add-Ins para Visual Studio, versión 11.1 Mod 1, Fixpack 1 se incluyen:

  • Microsoft Visual Studio 2010, Microsoft Visual Studio 2012, Visual Studio 2013, Visual Studio 2015 o Visual Studio 15
  • IBM Data Server Provider para .NET o IBM DB2 Data Provider para .NET Framework 2.0 y posteriores
  • Windows 7, Service Pack 2 o posterior; Windows Server 2008, Service Pack 2; o Windows Server 2008 R2; Windows 8 o posterior; Windows Server 2012; Windows 10.

Nota:  Versiones diferentes de los proveedores de datos dan soporte a versiones diferentes de Visual Studio. Los add-ins no se instalan para una versión instalada de Visual Studio salvo que haya instalada una versión correspondiente de un proveedor de datos.

  • IBM DB2 Data Provider for .NET Framework 2.0:
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 2.0, 3.0 o 3.5:
    • Visual Studio 2008
  • IBM Data Server Provider para .NET 4.0 o 4.5:
    • Visual Studio 2010
    • Visual Studio 2012
    • Visual Studio 2013
    • Visual Studio 2015
    • Visual Studio 15

Características clave de IBM Database Add-Ins for Visual Studio, Versión 11.1 Mod 1, Fixpack 1

Las nuevas características de IBM Database Add-Ins for Visual Studio, Versión 11.1 Mod 1, Fixpack 1 permiten a los desarrolladores reducir el tiempo de desarrollo y desarrollar aplicaciones .NET para toda la familia de servidores DB2, así como para Informix® Dynamic Server, utilizando Microsoft Visual Studio como su entorno de desarrollo integrado. La lista siguiente resume las características nuevas:

  • Soporte para Visual Studio 2015

IBM Database Add-Ins for Visual Studio, Versión 11.1 Mod 1, Fixpack 1 da soporte a los servidores de base de datos siguientes:

  • DB2 para Linux, UNIX y Windows Versión 9.7, y posteriores
  • DB2 para z/OS versión 10 y posteriores
  • DB2 para i V6R1, V7R1 y V7R2
  • Informix Dynamic Server, versión 11.5, versión 11.70 y versión 12.10
La disponibilidad de características y la compatibilidad de tipos de datos depende del servidor de datos que utilice. Por ejemplo, Informix Dynamic Server no da soporte al tipo de datos XML.

Nota:  Pueden aparecer problemas cuando la versión, el release y el nivel de fixpack de IBM Database Add-Ins for Visual Studio y un paquete de cliente no coinciden. Para asegurarse de que las funciones de las herramientas no den problemas, instale la misma versión, release y fixpack de los add-ins y uno de los paquetes de cliente siguientes:

  • IBM Data Server Client
  • IBM Data Server Runtime Client
  • IBM Data Server Driver Package para IBM Database Add-Ins

Soporte

El soporte para IBM Database Add-Ins for Visual Studio, Versión 11.1 Mod 1, Fixpack 1 se proporciona en el foro developerWorks® .NET. Este foro está supervisado por los desarrolladores de IBM Database Add-Ins for Visual Studio, IBM Data Server Provider para .NET e IBM DB2 Data Provider para .NET Framework 2.0 y posteriores. Este equipo de desarrolladores de IBM pretende interaccionar con los usuarios y está muy interesado en las respuestas o sugerencias de los mismos para incluirlas en los sucesivos releases de las herramientas de add-ins y proveedor de datos de IBM para Visual Studio y .NET.

Contribuya al éxito de esta forma de comunicación y envíe sus aportaciones a los desarrolladores de IBM participando activamente en el foro de developerWorks .NET.
