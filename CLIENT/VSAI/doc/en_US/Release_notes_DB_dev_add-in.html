IBM Database Add-Ins for Visual Studio, Version 11.1 Mod 1, Fix Pack 1 supports the IBM Data Server Provider for .NET and the IBM DB2 Data Provider for .NET Framework 2.0 and later.

Prerequisites

Prerequisites for the IBM Database Add-Ins for Visual Studio, Version 11.1 Mod 1, Fix Pack 1 include:

  • Microsoft Visual Studio 2010, Microsoft Visual Studio 2012, Visual Studio 2013, Visual Studio 2015 or Visual Studio 15
  • IBM Data Server Provider for .NET or IBM DB2 Data Provider for .NET Framework 2.0 and later
  • Windows 7, Service Pack 2, or later; Windows Server 2008, Service Pack 2; or Windows Server 2008 R2; Windows 8 or later; Windows Server 2012; Windows 10.

Note:  Different versions of the data providers support different versions of Visual Studio. The add-ins are not installed for an installed version of Visual Studio unless a corresponding version of a data provider is installed.

  • IBM DB2 Data Provider for .NET Framework 2.0:
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 2.0, 3.0, or 3.5:
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 4.0, or 4.5:
    • Visual Studio 2010
    • Visual Studio 2012
    • Visual Studio 2013
    • Visual Studio 2015
    • Visual Studio 15

Key features in IBM Database Add-Ins for Visual Studio, Version 11.1 Mod 1, Fix Pack 1

The new features in the IBM Database Add-Ins for Visual Studio, Version 11.1 Mod 1, Fix Pack 1 enable developers to reduce development time and develop .NET applications for the entire DB2 server family, as well as for Informix® Dynamic Server, by using Microsoft Visual Studio as their integrated development environment. The following list summarizes the new features:

  • Support for Visual Studio 15

The IBM Database Add-Ins for Visual Studio, Version 11.1 Mod 1, Fix Pack 1 supports the following database servers:

  • DB2 for Linux, UNIX, and Windows Version 9.7, and later
  • DB2 for z/OS Version 10 and later
  • DB2 for i V6R1, V7R1 and V7R2
  • Informix Dynamic Server, Version 11.5, Version 11.70 and Version 12.10
Availability of features and data type compatibility depends on the data server that you use. For example, Informix Dynamic Server does not support the XML data type.

Note:  Problems can occur when the version, release, and fix-pack level of the IBM Database Add-Ins for Visual Studio and a client package are not identical. To ensure that tooling functions work correctly, install the same version, release, and fix-pack level of the add-ins and one of the following client packages:

  • IBM Data Server Client
  • IBM Data Server Runtime Client
  • IBM Data Server Driver Package for IBM Database Add-Ins
