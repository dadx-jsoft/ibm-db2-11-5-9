IBM Database Add-Ins for Visual Studio

A partir de DB2 Bondi version 10.5.0.5, le support pour les serveurs de données IBM avec Microsoft Visual Studio 2008 devient obsolète.

Notes sur l'édition : Version 11.1 Mod 1, Fix Pack 1

© Copyright International Business Machines Corporation 2002, 2016. All rights reserved.

IBM Database Add-Ins for Visual Studio, Version 11.1 Mod 1, Fix Pack 1 prend en charge IBM Data Server Provider for .NET et IBM DB2 Data Provider for .NET Framework 2.0 et versions ultérieures.

Configuration requise

La configuration requise pour IBM Database Add-Ins for Visual Studio, Version 11.1 Mod 1, Fix Pack 1 comprend :

  • Microsoft Visual Studio 2010, Microsoft Visual Studio 2012, Visual Studio 2013, Visual Studio 2015 ou Visual Studio 15
  • IBM Data Server Provider for .NET ou IBM DB2 Data Provider for .NET Framework 2.0 et versions ultérieures
  • Windows 7, Service Pack 2 ou version ultérieure ; Server 2008, Service Pack 2 ; ou Windows Server 2008 R2 ; Windows 8 ou version ultérieure ; Windows Server 2012 ; Windows 10.

Remarque :  Les versions différentes des fournisseurs de données prennent en charge différentes versions de Visual Studio. Les extensions ne sont pas installées avec une version installée de Visual Studio, excepté si la version correspondante d'un fournisseur de données est installée.

  • IBM DB2 Data Provider for .NET Framework 2.0 :
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 2.0, 3.0 ou 3.5 :
    • Visual Studio 2008
  • IBM Data Server Provider for .NET 4.0 ou 4.5 :
    • Visual Studio 2010
    • Visual Studio 2012
    • Visual Studio 2013
    • Visual Studio 2015
    • Visual Studio 15

Principales fonctions d'IBM Database Add-Ins for Visual Studio, Version 11.1 Mod 1, Fix Pack 1

Les nouvelles fonctions d'IBM Database Add-Ins for Visual Studio, Version 11.1 Mod 1, Fix Pack 1 permettent aux développeurs d'écourter le temps de développement et de développer des applications .NET pour toute la famille de serveurs DB2, ainsi que pour les serveurs Informix® Dynamic Server, en utilisant Microsoft Visual Studio comme environnement de développement intégré. La liste suivante résume les nouvelles fonctions :

  • Prise en charge de Visual Studio 15

IBM Database Add-Ins for Visual Studio, Version 11.1 Mod 1, Fix Pack 1 prend en charge les serveurs de base de données suivants :

  • DB2 for Linux, UNIX, and Windows version 9.7 et version ultérieure
  • DB2 for z/OS version 10 et version ultérieure
  • DB2 for i V6R1, V7R1 et V7R2
  • Informix Dynamic Server version 11.5, version 11.70 et version 12.10
La disponibilité des fonctions et la compatibilité des types de données dépendent du serveur de données que vous utilisez. Par exemple, Informix Dynamic Server ne prend pas en charge le type de données XML.

Remarque :  Vous pouvez rencontrer des problèmes lorsque la version, l'édition et le niveau du groupe de correctifs d'IBM Database Add-Ins for Visual Studio et d'un module client sont différents. Pour être certain que les fonctions outils fonctionnent correctement, installez la même version et édition et le même niveau de groupe de correctifs que les modules complémentaires ainsi que l'un des modules client suivants :

  • IBM Data Server Client
  • IBM Data Server Runtime Client
  • IBM Data Server Driver Package for IBM Database Add-Ins
